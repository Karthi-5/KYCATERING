# K.Y Catering Service

## Overview

K.Y Catering Service is a modern, bilingual (English/Tamil) catering business website designed to showcase services, menus, and facilitate customer inquiries. The site features a visually appealing interface with gradient designs, floating animations, and responsive navigation including a mobile hamburger menu.

## User Preferences

Preferred communication style: Simple, everyday language.

## System Architecture

### Frontend Architecture

**Technology Stack**: Vanilla HTML/CSS/JavaScript with Vite as the build tool and development server.

**Key Features**:
- Single-page application with smooth scrolling navigation
- Section-based layout (#home, #about, #services, #menu, #gallery, #contact)
- Responsive design with mobile hamburger menu
- Bilingual content support (English and Tamil) using Google Fonts (Poppins and Noto Sans Tamil)
- Menu tab switching functionality with 4 different dinner menus
- Floating animations and gradient effects throughout

**Visual Design System**:
- CSS custom properties for consistent theming
- Primary gradient: Orange to gold (#ff6b35 to #ffd700)
- Floating animation keyframes for visual interest
- Glass-morphism design elements
- Mobile-first responsive approach

### Build System

**Vite Configuration**: Custom server settings optimized for development.

**Key Settings**:
- Host: 0.0.0.0 (accessible from external networks)
- Port: 5000 (strict port enforcement)
- Allowed hosts enabled for Replit compatibility
